# linux-games-fix-scripts
A collection of scripts used to patch some games that wouldn't normally work on Linux

This repository was inspired by the event described in: https://bsky.app/profile/morg.systems/post/3miyun3cewc26

Turns out LLMs are actually pretty fucking good at randomly fixing bugs in binaries and live patching old games that wouldn't normally work under wine on Linux. 

So I thought, why not just gather a bunch of these patching scripts? Thought it might be fun.

NOTE: This repository doesn't contain binaries, you will have to get your own binaries.

Each directory references a different game, including links where the game was legally obtained (if possible), sha512 signature of the pre-patch binary, and a script to patch said binary to make it work. Plus any additional comment.
